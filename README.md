# -D-E-T-E-R-M-I-N-I-S-T-I-C-R-E-F-L-E-C-T-O-R-
Deterministic Reflector is a rule-based decision support system that provides consistent daily guidance using inputs like mood, energy, time, and productivity. It features a decision trace for transparency, input validation, and session logging, ensuring reliable, explainable, and structured recommendations.
